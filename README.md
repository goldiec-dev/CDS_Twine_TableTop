# CDS Twine Table Top Excercise 
Interactive, decision-based cybersecurity awaremess game. Players assume different organizational roles (Intern, IT Department, Mangement, Red Hat)  and navigate real-world cyber senarios where their choices effect the outcome. 

## Editing Twee File
1) Download Tabletop.twee file from this repository
2) go to https://twinery.org/2/#/ and then go to library > import
