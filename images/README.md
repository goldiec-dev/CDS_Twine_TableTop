# Images Folder

## Adding Background to Twee File
1. When in Twinery, click on the desision point that you want to change from the default background.
2. when the text shows up on the right click the 'Tags' button
3. In the 'Tag Name:' textbox type one of these options: Redhat, Intern, IT, UM
## Adding Image to Twee File
```
<div class="centered-image">
<img src="https://raw.githubusercontent.com/goldiec-dev/CDS_Twine_TableTop/main/images/<IMAGE_NAME>" style ="width:50%";>
</div>
```
